# 勤怠記録PWA 設計書

**作成日**: 2026-06-14  
**プロジェクト**: `/Users/apple/Desktop/Levtech`

---

## Context

毎日の始業・終業時刻をスマホで手軽に記録し、月末にLevtech勤怠報告サイトへ一括反映することを目的とする。既存のシンプルなPWA（`index.html`）を拡張して実装する。

---

## アーキテクチャ

```
📱 PWA（スマホ・Safari ホーム画面追加）
  ↕ fetch（JSON）
⚙️ Google Apps Script（GAS）── Web API サーバー
  ↕ Sheets API
📊 Googleスプレッドシート ── データストア（月ごとにシート）
  ↓ ブックマークレット（PCブラウザで実行）
🌐 Levtech 勤怠入力ページ（月ごとにURLが変わる）
```

---

## 画面構成（3タブ）

### タブ①「今日」
- 今日の日付・曜日を表示
- 始業ボタン・終業ボタン（タップで現在時刻を15分単位に切り上げて記録）
- 今日の記録済み始業・終業時刻を表示
- 記録済みの場合は時刻をタップして修正可能

### タブ②「カレンダー」
- 月カレンダー表示（前月・翌月へ矢印ナビ）
- 記録済み日にはドット表示
- カレンダー下部に直近の記録リストを表示
- 各行の✏️アイコンをタップ → 時刻編集モーダルを表示
- 土日は非稼働日として薄く表示（記録・警告の対象外）

### タブ③「Levtech」
- **今月の入力URL欄**：URLを貼り付けて保存（月ごとに変更可。GASに保存）
- **記録サマリー**：記録済み日数／営業日数（平日のみ・祝日除く）、平均始業・終業
- **未記録警告**：平日・非祝日のみカウント。未記録日がある場合のみ表示
- **ブックマークレットをコピー**ボタン：月の全記録を埋め込んだコードをクリップボードへ
- 操作説明：「PCでLevtechを開いてから実行してください」

## データ設計

### Googleスプレッドシート構造
- シート名：`2025-06`（年月形式、月変わりで自動作成）
- 列：`date`（YYYY-MM-DD） | `start`（HH:MM） | `end`（HH:MM） | `note`
- 別シート `settings`：`levtech_url_YYYY-MM`（月ごとのURL）を保存

### GAS API エンドポイント（Web App としてデプロイ）
| メソッド | パラメータ | 処理 |
|---------|-----------|------|
| `GET` | `month=YYYY-MM` | 該当月の全記録を返す |
| `POST` | `date, start, end` | 1件保存・更新（upsert） |
| `GET` | `action=settings&month=YYYY-MM` | LevtechURLを返す |
| `POST` | `action=settings&month=YYYY-MM&url=...` | LevtechURLを保存 |

### localStorage（オフライン対応）
- GASへの通信失敗時はlocalStorageにキャッシュ
- 次回オンライン時に自動同期

---

## ブックマークレット（月次一括入力）

PCブラウザでLevtech勤怠入力ページを開いた状態で実行すると：
1. GAS APIから取得した当月の全記録データをコード内に埋め込み済み
2. ページ内の全行をMM/DD形式で走査
3. 該当行の始業・終業inputに値をセット（Reactイベント発火対応）
4. 完了後にアラートで入力件数を表示

---

## 祝日対応

- 日本の祝日を静的JSONとして保持（年度ごと）、または [内閣府CSVフィード](https://www8.cao.go.jp/chosei/shukujitsu/syukujitsu.csv) から初回取得してlocalStorageにキャッシュ
- 祝日・土日は「未記録警告」「営業日カウント」から除外

## 検証方法

1. **GASデプロイ確認**：`Code.gs` をGASに貼り付け、Web Appとしてデプロイ。curlでGET/POSTを叩いてデータが書き込まれることを確認
2. **PWA動作確認**：iPhoneのSafariでページを開き「ホーム画面に追加」→ 始業・終業ボタンを押してスプレッドシートに記録されることを確認
3. **カレンダー編集確認**：カレンダーの✏️から時刻を編集し、GAS経由でスプレッドシートが更新されることを確認
4. **ブックマークレット確認**：PCでLevtechページを開き、コピーしたブックマークレットを実行。全行に時刻が入力されることを確認
5. **月切り替え確認**：月が変わった際にスプレッドシートに新しいシートが自動作成されることを確認
